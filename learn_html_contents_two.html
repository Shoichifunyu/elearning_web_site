学習内容①

HTMLのタグには様々な種類がある。

ここでは、代表的なタグとその使い方を示す。

タグの要素 名称 使い方
<a> アンカー要素 <a>単独で使用するのではなく、<a href=>で別のウェブページ、ファイル、メールアドレス、同一ページ内の場所、または他の URL へのハイパーリンクを作成する。
<h1>、<h2>、<h3>、<h4>、<h5>、<h6> HTML の見出し要素 HTML の <h1>–<h6> 要素は、セクションの見出しを 6 段階で表す。<h1> が最上位で最も見出しが大きく、<h6> が最下位で最も見出しが小さい。
<img> 画像埋め込み要素 文書に画像を埋め込む。
